# API Stripe Client ID

## Overview

- This is a project where I practice integrating Stripe API using Next.js. The main purpose is to learn how to handle payment requests by calling Stripe with a Client ID. This project helps improve my skills in building backend API routes and securely managing payment flows on the frontend.
- In addition, this project also includes a API that provides data for provinces, districts, and wards in Vietnam (e.g., /api/provinces, /api/provinces/districts/[provinceCode]).
